# pean-client

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.0.7.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Docker container

Run `docker-compose up` to run in a Docker container. Navigate to `http://localhost:4200/`. The app will NOT automatically reload if you change any of the source files. This is how the app we be deployed in production.

## Backend Server

Note that the app expects that a backend server will be running on the local host on port 3000 by default. In Docker, the backend server will be on an internal network named `pean-server_default` and hostname `backend`.
